# Creative AI Plugin

AI Video Production Suite for Claude Code.

## Skills

### seedance-director
Converts scene descriptions into production-ready bilingual EN+ZH prompts for Seedance 2.0.
3-step character pipeline, physics-first prompting, camera behavior direction.

### ai-series-director
Full Zephyr-level production pipeline for multi-character episodic AI series.
Character creation, world building, scene direction, K-Pop formations.

## Install
```
claude plugin add /path/to/creative-ai-plugin
```
